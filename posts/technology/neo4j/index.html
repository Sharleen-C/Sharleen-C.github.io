Neo4j 教程#

启动#

通过使用neo4j.bat文件启动服务器，该文件存放在主目录的bin文件夹中

在cmd中，输入neo4j.bat console并回车

打开：localhost:7474

账号：neo4j

密码：neo4j2023

Neo4j CQL#

CREATE语法#

1CREATE (<node-name>:<label-name>)
2// node-name：节点名称；label-name：节点标签名称
3
4CREATE (<node-name>:<label-name1>:<label-name2>.....:<label-namen>)
5// 多个标签
6
7CREATE (<node1-name>:<label1-name>)-[<relationship-name>:<relationship-label-name>]->(<node2-name>:<label2-name>)
8// 单个标签到关系
  • Neo4j使用此将此节点详细信息存储在Database.As中作为Neo4j DBA或Developer，我们不能使用它来访问节点详细信息。
  • Neo4j创建一个作为内部节点名称的别名。作为Neo4j DBA或Developer，我们应该使用此==标签名称==来访问节点详细信息。
CREATE (emp:Employee)

创建具有属性的节点，通过创造以键值对表示的属性节点来存储数据

1CREATE (
2    <node-name>:<lable-name>
3    {
4    	<Property1-name>:<Property1-Value>
5    	.........
6    	<PropertyN-name>:<PropertyN-Value>
7    }
8)

Example：

1CREATE (
2    dept:Dept{
3		deptno:10,dname:"Accounting",location:"Hyderabad"    
4    }
5)
6
7// 创建关系
8CREATE (p1:Profile1)-[r1:LIKES]->(p2:Profile2)

MATCH　指令#

1MATCH (<node-name>:<label-name>)

不可单独使用，需配合RETURN子句或其他子句使用

RETURN 子句#

用于检索节点的某些属性、所有属性、节点和关联关系的某些属性、所有属性

1RETURN 
2	<node-name>.<property1.name>,
3	.........
4	<node-name>.<propertyn-name>

MATCH & RETURN匹配和返回#

1MATCH Command RETURN Command

示例

1MATCH (dept:Dept)
2RETURN dept.deptno,dept.dname
3
4MATCH (dept:Dept)
5RETURN dept

WHERE 子句#

1WHERE <condition> <boolean-operator> <condition>

<condition>语法

1<property-name> <comparison-operator> <value>
2
3//<comparison-operator>比较运算符
运算符 解释
AND 与
OR 或
NOT 非
XOR 异或
= 等于
<> 不等于
< 小于
> 大于
<= 小于等于
>= 大于等于 
1MATCH (emp:Employee) 
2WHERE emp.name = 'Abc'
3RETURN emp

使用WHERE 子句创建关系

1MATCH (<node1-label-name>:<node1-name>),(<node2-label-name>:<node2-name>) 
2WHERE <condition>
3CREATE (<node1-label-name>)-[<relationship-label-name>:<relationship-name>
4       {<relationship-properties>}]->(<node2-label-name>) 

1//example
2MATCH (cust:Customer),(cc:CreditCard) 
3WHERE cust.id = "1001" AND cc.id= "5001" 
4CREATE (cust)-[r:DO_SHOPPING_WITH{shopdate:"12/12/2014",price:55000}]->(cc) 
5RETURN r

DELETE 子句#

1MATCH (e: Employee) DELETE e

1DELETE <node1-name>,<node2-name>,<relationship-name>

1//删除两个节点及其关联关系
2MATCH (cc: CreditCard)-[rel]-(c:Customer) 
3DELETE cc,c,rel

REMOVE子句#

用于删除节点或关系的标签或属性

DELETE REMOVE
区别 用于删除节点和关联关系 用于删除标签和属性
相似 都不应单独使用 都应该与MATCH命令一起使用 
1REMOVE <property-name-list>
2
3// <property-name-list>语法
4<node-name>.<property1-name>,
5<node-name>.<property2-name>, 
6.... 
7<node-name>.<propertyn-name>

Example

1CREATE (book:Book {id:122,title:"Neo4j Tutorial",pages:340,price:250}) 
2
3MATCH (book : Book) RETURN book
4
5MATCH (book { id:122 }) REMOVE book.price RETURN book

删除节点/关系的标签

1REMOVE <label-name-list> 
2
3//<label-name-list>语法
4<node-name>:<label2-name>, 
5.... 
6<node-name>:<labeln-name>
