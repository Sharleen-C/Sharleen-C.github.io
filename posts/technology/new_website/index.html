关于页面展示#

  1. 在content文件夹中，新增博客文件，除了archive和search直接建在content文件夹下，其他内容需要首先新建文件夹
  2. 新建文件夹中需要新增_index.md文件，形成层级标题
  3. 文件夹层层嵌套，形成层层标题
  4. archetypes/default.md中包含的是每次新增一个页面遵循的模板，如下
 1---
 2title: "My post"
 3date : '{{ .Date }}' # 形成Archive所必需的
 4lastmod : '{{ .Date }}'
 5# weight: 1
 6tags: ["first"]
 7author: "Sharleen"
 8showToc: true # md Toc
 9TocOpen: true
10draft: false
11hidemeta: false
12comments: false
13description: "Desc Text."
14canonicalURL: "https://canonical.url/to/page"
15disableHLJS: true # to disable highlightjs
16disableShare: false
17hideSummary: false
18searchHidden: false #设置为true，Search页面搜索不到
19ShowReadingTime: true
20ShowBreadCrumbs: true
21ShowPostNavLinks: true
22ShowCodeCopyButtons: true
23cover:
24    image: "<image path/url>" # image path/url
25    alt: "<alt text>" # alt text
26    caption: "<text>" # display caption under cover
27    relative: false # when using page bundles set this to true
28    hidden: true # only hide on current single page
29---

关于部署#

按照 建站方法 中的方法进行构建即可。

参考：Hugo博客从本地托管到github

期间，可能会遇到一个问题！那就是本地hugo server时界面正常显示，但是github部署之后界面仅保留文字，没有渲染CSS / JS。 此时对界面检查会发现一个报错：

Failed to find a valid digest in the 'integrity' attribute......

通过查询官方文档可以解决该问题：官方解答

在hugo.toml中增加如下部分：

1[params.assets]
2disableFingerprinting = true

重新部署即可。

关于 Git#

只需要将public文件夹同步至Github上即可，因此先转到public文件下，进行初始化和与远程repository进行连接

1cd ./public
2git init -b main
3git remote add origin https://github.com/Sharleen-C/Sharleen-C.github.io.git

！注意！：在commit和push之前养成好习惯，先pull!

1git pull https://github.com/Sharleen-C/Sharleen-C.github.io.git main

将public文件夹下的所有文件commit

1git add .
2git commit -m "v1"

向远程仓库push，注意此处代表branch为main，如果远程仓库branch为master，记得修改！

1git push origin main

push的另外一种方式

1git push --set-upstream origin main

方法二：适用于特殊字体#

JetBrains Mono 最漂亮的编程字体

  1. 下载字体
  2. 将下载好的字体放到static/fonts文件夹中
  3. 在assets/css/extended中新增文件fonts.css
1@font-face {
2font-family: "JetBrainsMono_Regular";
3src: url("/fonts/JetBrainsMono-Regular.woff2 ") format("truetype");
4}
  1. 在assets/css/common/post-single.css中修改
 1.post-content code {
 2    margin: auto 4px;
 3    padding: 4px 6px;
 4    font-size: 0.78em;
 5    letter-spacing: 20px;
 6    font-family: JetBrainsMono-Regular;
 7    line-height: 2;
 8    background: var(--code-bg);
 9    border-radius: 2px;
10}
