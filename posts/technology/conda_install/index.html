Conda安装以及创建虚拟环境#

conda安装#

首先查看自己是否安装了conda，键盘Win+R, 输入cmd，按回车，输入conda -V，查看版本号。

下载Anaconda,直接搜索，进入官网， 依次点击，选择与自己电脑对应的版本，进行下载。

右击，以管理员身份运行，进行安装。

默认安装路径D:\Anaconda\install

一定要勾选添加环境变量！点击Install，安装

这里两个方框都不勾选，防止打开其他网页。

查看Anaconda是否安装成功

Anaconda创建Pytorch虚拟环境#

首先使用conda创建虚拟环境#

输入activate，激活，进入base环境。

输入conda create -n pytorch1 python=3.8 #指定python版本号创建名为pytorch1虚拟环境

创建完成

base环境输入conda activate pytorch1，进入创建的虚拟环境。并验证python环境

在PyCharm中配置Interpreter

File-Settings-Project-Project Interpreter-Add Interpreter

conda info --envs查看是否有虚拟环境

conda install pytorch torchvision torchaudio cpuonly -c pytorch安装pytorch

Conda创建等指令#

创建虚拟环境

1conda create -n xxx python=3.?

1source activate xxx
2# or
3conda activate xxx

退出虚拟环境

1conda deactivate

删除虚拟环境

1conda remove -n xxx --all

查看虚拟环境

1conda info --env

其他问题#

  1. Conda cannot proceed due to an error in your proxy configuration. Check for typos and other configuration errors in any ‘.netrc’ file in your home directory, any environment variables ending in ‘_PROXY’, and any other system-wide proxy configuration settings.

    该问题源于网络问题，可以打开/关闭VPN试试

    【已解决】ProxyError: Conda cannot proceed due to an error in your proxy configuration.Check for typos a-CSDN博客

  2. AttributeError: module numpy has no attribute object . np.object

    numpy版本太高导致的，最新版本通常会改很多东西，直接降低numpy版本就行，一个版本一个版本慢慢试下去就行

    conda uninstall numpy

    conda install numpy==1.20.0

  3. PackagesNotFoundError: The following packages are not available from current channels:

    PackagesNotFoundError: The following packages are not available from current channels的解决办法-CSDN博客

    PackagesNotFoundError: The following packages are not available from current channels 解决办法-CSDN博客

    （清华镜像不能用，改用豆瓣或阿里）

  4. 【陆续排坑】解决conda下载速度慢的问题：更换国内源_conda config –remove-key channels-CSDN博客
